# Blog JSON

## Persiapan

Install Node JS

```bash
sudo apt install node

# Gunakan Node yang versi 12 atau 13 awal
# Soalnya yang versi 13 baru, kadang ada errornya
```

Install JSON Server

```bash
npm i -g json-server
```

## Menjalankan

```bash
./jalankan
```

## Mengedit markup

Install Pug

```bash
npm i -g pug-cli
```

Jalankan script

```bash
./olah
```
